Propriedade de borda abreviada CSS

Como você viu na página anterior, há muitas propriedades a serem consideradas ao lidar com bordas.

Para encurtar o código, também é possível especificar todas as propriedades de borda individuais em uma propriedade.

A propriedade border é uma propriedade abreviada para as seguintes propriedades de borda individuais:

  • border-width: largura da borda.
  • border-style(obrigatório): estilo da borda.
  • border-color: cor da borda.

Todas as bordas

   Exemplo CSS
   Todas as bordas com largura de 10px, estilo sólido 
   e cor vermelha
   p#one { 
      border: 10px solid red;
      padding: 10px;
   }

Resultado do Exemplo Acima

   
Este é um parágrafo com bordas sólidas de 10px vermelhas.

Borda Esquerda

Pode-se especificar todas as propriedades de borda individuais para apenas um lado:

>
   Exemplo CSS
   Borda Esquerda com 6px de largura, estilo sólido 
   na cor vermelha
   p#two {
      border-left: 6px solid red;
      padding: 10px;
   }

Resultado do Exemplo Acima

   
Este é um parágrafo com a borda Esquerda sólida de 6px e vermelha.

Borda Inferior

      Exemplo CSS
      Borda inferior com 6px de largura, sólida e vermelha
      p#three {
         border-bottom: 6px solid red;
         padding: 10px;
   }

Resultado do Exemplo Acima

   
Este é um parágrafo com a borda Inferior sólida de 6px vermelha.

Bordas Arredondadas

   Exemplo CSS
   Todas as bordas com largura de 10px, estilo sólido 
   e cor vermelha
   p#radio { 
      border: 10px solid red;
      border-radius: 20px; /* Arredonda as bordas 20px */
      padding: 10px; /* espaço interno de 10px */
   }

Resultado do Exemplo Acima

   
Este é um parágrafo com bordas arredondadas, 
      sólidas de 10px na cor vermelha.
